# Bases de datos documentales

Este tipo de base de datos almacena la información en documentos con formato similar a JSON. Cada documento puede tener su propia estructura.

Permiten representar datos complejos de forma flexible y cercana al modelo utilizado en aplicaciones modernas.

Ventajas:

* Alta flexibilidad
* Facilidad para evolucionar el modelo
* Buen soporte para consultas

Desventajas:

* Posible duplicación de información
* Relaciones complejas pueden ser costosas

Casos de uso comunes:

* Aplicaciones web
* Sistemas de contenido
* Comercio electrónico

Ejemplos: MongoDB, CouchDB, Firestore.
